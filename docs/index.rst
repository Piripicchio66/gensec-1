Welcome to GenSec's documentation!
==================================

GenSec is a modular Python program for fiber-based cross-section 
analysis, mainly of reinforced concrete structures. The project 
targets combined axial force and bending (presso-tensoflessione), 
with a roadmap extending to biaxial bending, 3D resistance surfaces, 
prestress, shear, and interface slip.

The script itself is developed in Python, under uv virtual environment.

.. important:: **The software has not yet been fully validated**
