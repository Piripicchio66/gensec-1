=========================
:mod:`gensec.io_yaml`
=========================

YAML input loader.

Reads a YAML file describing materials, section geometry, and
(optionally) load demands, and returns fully constructed GenSec
objects ready for analysis.  Supports both the legacy rectangular
format and the generic section format with parametric shapes or
custom polygons.

See :doc:`/user_guide/yaml_reference` for the full input specification.
